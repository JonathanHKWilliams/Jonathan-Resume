# Project Images

Add your images in this folder with the following filenames:

## Profile Banner Image:
- `profile-banner.jpg` - Your professional headshot for the top banner (recommended: square format, high quality)

## Project Screenshots:
- `karnue-project.jpg` - Screenshot for Karnue Ride Sharing Platform
- `nemis-project.jpg` - Screenshot for NEMIS Education Dashboard  
- `electpal-project.jpg` - Screenshot for ElectPal School Election Platform
- `growlocal-project.jpg` - Screenshot for GrowLocal Farmers Marketplace

## Design Tool Logos:
Place these in the `/images/` folder (same folder as other images):
- `figma-tool.jpg` - Figma logo
- `photoshop-tool.jpg` - Adobe Photoshop logo
- `illustrator-tool.jpg` - Adobe Illustrator logo
- `xd-tool.jpg` - Adobe XD logo
- `canva-tool.jpg` - Canva logo

## Image Guidelines:

### Profile Banner:
- **Format**: Full-side banner image (takes up left half of banner)
- **Recommended size**: 1200x800 pixels or higher for best quality
- **Aspect ratio**: Any landscape format will work well
- **Quality**: High resolution, professional photo
- **Style**: Professional appearance that fills the space effectively

### Project Screenshots:
- **Recommended size**: 1200x800 pixels or similar aspect ratio
- **Format**: JPG, PNG, or WebP
- **File size**: Keep under 500KB for fast loading
- **Content**: Show the best view of your project's interface

### Design Tool Logos:
- **Format**: PNG with transparent background (recommended)
- **Size**: 200x200 pixels or higher for best quality
- **Background**: Transparent or white background works best
- **Style**: Official logos or high-quality representations

## How it works:
- Images will be displayed automatically when you add them with the correct filenames
- If an image is missing or fails to load, emoji fallbacks will be shown
- Profile image uses circular crop with purple border
- Project images use `object-cover` to maintain proper aspect ratio
- Tool logos display in clean cards with fallback text

## Adding your images:
1. Save your images with the exact filenames listed above
2. Place them in the appropriate folders (`/public/images/` or `/public/images/tools/`)
3. Restart your development server if it's running
4. Refresh your browser to see the images

The fallback emojis will ensure your portfolio looks good even before you add the real images.
